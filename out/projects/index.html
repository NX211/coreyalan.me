Traefik WebFinger Plugin

GoTraefikWebFinger

A Traefik plugin that implements the WebFinger protocol (RFC7033), enabling federated identity for web applications. This plugin allows Traefik to respond to WebFinger requests, which is crucial for ActivityPub and other federated protocols.

Traefik Proxmox Provider

GoTraefikProxmox

A Traefik provider for Proxmox VE that automatically discovers and routes traffic to virtual machines and containers. This plugin simplifies network configuration in Proxmox environments by dynamically generating Traefik routes.
